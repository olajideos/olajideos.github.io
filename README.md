# Olajide Olamide Shobowale - Professional Portfolio Website

A clean, professional static website showcasing career achievements, projects, and publications.

## Features

- Responsive design that works on desktop, tablet, and mobile devices
- Clean, minimalist aesthetic with dark theme
- Comprehensive career history and achievements
- Project showcase with detailed descriptions
- Publications and thought leadership articles
- Awards, certifications, and professional memberships
- SEO-friendly HTML structure
- Print-friendly CSS

## Files Included

- `index.html` - Main HTML file with all content
- `styles.css` - Stylesheet with responsive design
- `README.md` - This file with deployment instructions

## Deployment Options

### Option 1: GitHub Pages (Recommended)

1. Create a new repository on GitHub named `olajideos.github.io`
2. Upload these files to the repository:
   ```
   git clone https://github.com/olajideos/olajideos.github.io.git
   cd olajideos.github.io
   # Copy index.html and styles.css to this directory
   git add .
   git commit -m "Initial portfolio website"
   git push origin main
   ```
3. Your site will be live at: `https://olajideos.github.io`

### Option 2: Traditional Web Hosting

1. Upload all files to your web hosting provider via:
   - FTP/SFTP client (FileZilla, Cyberduck, etc.)
   - cPanel File Manager
   - Your hosting provider's control panel

2. Ensure files are in the public root directory (usually `public_html`, `www`, or `htdocs`)

3. Access your site at your domain: `https://yourdomain.com`

### Option 3: Netlify (Easy & Free)

1. Visit [Netlify](https://www.netlify.com)
2. Drag and drop the entire folder into Netlify's deploy zone
3. Your site will be live instantly with a free subdomain
4. Optional: Connect your custom domain

### Option 4: Vercel

1. Visit [Vercel](https://vercel.com)
2. Import your GitHub repository or upload files directly
3. Deploy with one click
4. Free hosting with custom domain support

## Local Development

To test locally:

1. Simply open `index.html` in any web browser
2. Or use a local server:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   
   # Node.js (if you have http-server installed)
   http-server
   ```
3. Visit `http://localhost:8000` in your browser
